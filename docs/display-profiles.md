# Display Profiles

La fonctionnalité Display Profiles a été retirée de l'interface et du flux principal.

WattPilot ne modifie pas HDR, VRR ou G-Sync, et n'affiche plus ces diagnostics dans le dashboard principal ni dans le tray. Les anciens réglages locaux liés aux profils écran peuvent rester présents dans un fichier `settings.json` existant, mais ils sont ignorés par les versions actuelles.
